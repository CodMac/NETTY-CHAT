2.自定义协议========================================================================= start

  bytes
  --------------
  |  |  |  |  |
  --------------
  4位		-- 魔数: 用于标注消息开头. 'M','A','C','!'
  1位 		-- 协议版本号. 0b00000001
  1位 		-- msgModule: 用于区分消息类型 
  4位 		-- 消息体长度
  n位	 	-- 消息体
